# PW1-ProyectoFinal

Proyecto final de la materia de programación web 1

## Contenido:
Una página web de notas.
* Inicio de sesión
* Registro de usuario
* Agregar, leer, actualizar y eliminar notas 
* Perfil de usuario
* Editar datos de usuario
* Busqueda de notas
* Etiquetas
* Validaciones
